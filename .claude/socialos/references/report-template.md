# Report Templates — SOCIAL:OS

Visual templates for social media performance reports. Used by `/social:report`.

---

## Report Types

- **Weekly social report** -- quick status, key metrics, top content
- **Monthly social report** -- full analysis, growth, performance by platform
- **Quarterly strategy review** -- strategic analysis, goals assessment, planning
- **Newsletter report** -- email-specific metrics
- **Campaign report** -- specific initiative performance
- **Competitor benchmark report** -- comparative analysis

---

## Notes

- All reports use the SOCIAL:OS box style from ui-brand.md
- Include benchmark comparisons from BENCHMARKS.md where relevant
- Highlight metrics above benchmark with [STRONG], below with [WEAK]
- Reports can be exported as markdown files to `performance/reports/`
- Always include actionable recommendations, not just data
