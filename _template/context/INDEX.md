# Context Index

This folder stores workspace-specific context files that SOCIAL:OS loads as needed.

## How It Works

Context files provide background information that shapes content creation, engagement, and strategy. They are loaded on demand -- not all at once.

## Standard Context Files

| File               | Purpose                                  | Load When                    |
|--------------------|------------------------------------------|------------------------------|
| brand-story.md     | Origin story, founder background         | Writing about-us content     |
| product-info.md    | Product/service details, features, pricing | Writing product posts       |
| case-studies.md    | Customer success stories                 | Creating social proof content|
| faq.md             | Common questions and answers             | Community engagement         |
| media-kit.md       | Press mentions, features, awards         | Authority-building content   |
| partnerships.md    | Partner brands, collaborations           | Co-marketing content         |
| events.md          | Upcoming events, webinars, launches      | Event promotion              |
| swipe-file.md      | Saved examples of great content          | Inspiration, content creation|

## Adding Context

1. Create a markdown file in this folder
2. Add it to the table above
3. Reference it in your content briefs

> Keep context files focused. One topic per file. Update when information changes.
